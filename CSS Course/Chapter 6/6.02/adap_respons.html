Adaptive design involves considering every possibility for the device where a website is going to be viewed on, and then creating CSS for each individual one. This might not be the answer because it is very tedious to code different CSS for the so many devices out in the world right now, and there might be more devices in the future that have different screen sizes.

Responsive design involves making the CSS code flexible so that it can shrink or get bigger on an as needed basis. This sounds great, but it is very hard to code CSS this way. Hence, it is best to use a combination of the two.

Tip: When creating websites, it is useful to think about some things: 1. On hover elements or On Click elements: This is easy to do on computers but might be hard for mobile devices. It is hard to click on something and impossible to hover on mobile devices. Also, one's finger might be blocking something that they are trying to read. 2.
