The border-style property can be toggled to get a style of border. Some examples are dashed, dotted, groove, double, and solid. You can also specify a color or colors with the border-color property. Just type the list of colors and the editor will add the colors in a clockwise format. You can use the border-radius property to make a rounded border look. Enter the pixel numnber and that is how many pixels the radius of the border circle will be.
