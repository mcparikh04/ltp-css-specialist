The box model is as follows: Padding, Border, Margin, then Outline.


Padding: You need four values to express the padding around an element and goes clockwise: Top, Right, Bottom Left. Ex. padding: 10px, 10px, 10px, 10px


Border: You need 3 values to express the padding around an element: Thickness, Style, and Color. Ex. border: 10px solid black.


Margin: You need four values to express the padding around an element and goes clockwise: Top, Right, Bottom Left. Ex. margin: 10px, 10px, 10px, 10px


Outline: You need 3 values to express the padding around an element: Thickness, Style, and Color. Ex. outline: 10px solid red.
