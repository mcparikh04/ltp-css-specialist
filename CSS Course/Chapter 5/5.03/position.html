We are looking at the position property in this video. We have looked at the satic attribute, which is the default. Static follows normal flow, which means it foolws the inline format (going left to right, finsihed in botton right.)

The offset property allows you to move the postioning of an element around. To write an offset, write the driection and then the amount of pixels wanted to be offset. Offset cannot work when the positioning is set to static, and static overrides.

The relative attribute works with offest. It follows normal flow, but shows in adifferent position. The element stil takes up the original space, but the text area is different.

When we put absolute for the text, the text shows 100px from the top of the screen or the body/parent element. Absolute atrribute does not follow normal flow.

The fixed attribute is a little different. The text looks exatly the same as absolute, but when we scroll the page, the text stays where it is and the rest scrolls regularly.


Ther are a few more atttributes that determine poistion. THe z-index determines how the elements are stacked on the page. For the z-index, either put a -1 one or positive 1. For our blue text, we put a z-index of -1 and the text showed behind the grey text. If we put a 1, it will show on top.

We can put percentages for the offset also for values. If we put the top at 50%, the top element will show 50% down from the parent element.
