For div elements, the default display is as a block element.

In this lesson, we are trying to make the grey text wrap around the red sqaure. We can easily do it with the float property, but we are going to try to use other ones to see if it will work.

If we try display as inline block, it will not work. The red square will be in the same line as the text on the line, but since its height is so large at 200px, it is making its whole line 200px tall. If we try display as absolute and have an offset of 0px from the left so it is stuck there, it will not work. There will be no more white space, but there will be text hiding behind it.

If we put the float attribute with the left value, it will do what we want. There will be no wasted white space and the grey text will completely wrap around.

We can make another box and add another block of css. If we put the float as right, the boxes will sit on either side. If we put both as float left, they will sit right next to each other and the text will still wrap around.
