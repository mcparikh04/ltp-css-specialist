In this example, we made a slider. This was an on hover element, and when we hovered our mouse over it, the slider made two changes; the width and color. We set a width of 0 pixels in the beginning(with padding so the slider can be seen) and a background color of grey.We then used the transition property to set a width and background color in seconds. This means that the computer will take that amount of seconds to change the width and background color on hover. We also used the timing function to slowly come out. Finally, we made a slider:hover selector that stated the final width of 80px and a backgoround color of pinkish red. You can toggle with the seconds to change the speed of the slider transition. Transitions can make a website come alive and be interactive!
